# [MISC] ふっかつのじゅもん - 500Point 2Solves

## 問題文 

魔王を倒してフラグを入手しよう！

ふっかつのじゅもんで何度でも蘇れるぞ！

セーブ方法

1. Tabキーを押してメニューを開く
2. セーブ を押す
3. じゅもんをコピー
4. ゲームを終了を押してタイトルに戻る
5. じゅもんを入力する
6. フラグフォーマットはipfctf{xxxxxx}です。
